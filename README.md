# Idea For Hackathon
Readify is a simple AI-powered tool that rewrites complex text into clear, dyslexia-friendly language with better spacing, fonts and readability. It also gives word meanings, quick quizzes and text-to-speech so students understand faster and learn with confidence.

## Member Duties
- **Prithvi** : Frontend (HTML+CSS)  
- **Vedant** : AI Integration and JS
- **Agrim** : Presenter

## Technologies/Programming Languages used
- HTML  
- CSS  
- JS  
- Gemini API

# Website Structure and Design

## Readify  
** Reading, But Better**

Readify is a simple, accessibility-focused web app created to help students with dyslexia read more comfortably. Built with a clean, Apple-style UI and a focus on clarity, accessibility, and ease of use.

# App Structure

## 1. Home Page
The Home page is the central hub of the app and the only visually complex page.

It contains links to:
- Dyslexia Simplifier  
- Awareness Page  
- About Page  
- Settings


---

## 2. Dyslexia Text Simplifier
This is the core feature designed to help people read.

### Workflow
- User pastes text   
- Clicks Simplify  
- Receives a cleaner, easier-to-read output with a dyslexia-friendly font

---

## 3. Awareness Page
A simple but impactful page that highlights the plight of hundreds of millions of people worldwide.
### Sections
- What dyslexia is  
- Common myths (busted)  
- Global impact statistics  
- Why accessibility matters  
- How Readify helps  

This page uses static text plus simple visuals like icons or large typography.
